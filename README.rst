Installation
^^^^^^^^^^^^

::
    $ pip install xiver-gpt

Installation from source
^^^^^^^^^^^^^^^^^^^^^^^^

Dependencies:
-------------

* **poetry**

:: 
    $ git clone https://github.com/xiver/xiver-gpt.git; cd xiver-gpt-master
    $ poetry install
    $ poetry run build; cd dist
    $ pip install $(ls -Art | tail -n 1)
